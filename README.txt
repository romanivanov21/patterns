          Реализация паттернов проектирования на языке С++
_____________________________________________________________________

                    ТРЕБОВАНИЯ К СБОРКЕ ПРОЕКТА
_____________________________________________________________________
Система:
    -OS Linux kernel version=3.16.0
Необходимо установить пакеты:
    -gcc 	version=4.9.3
    -g++ 	version=4.9.3 ( поддержка последнего стандарта с++14 )
    -cmake 	version=2.8.12.2
    -dia	version=0.97.2
______________________________________________________________________

                     ОПИСАНИЯ СТРУКТУРЫ ПРОЕКТА
______________________________________________________________________
Общие папки:
~/patterns/                                     |-папка проекта
~/--------/bin/                                 |-папка для исполняемых 
                                                  файлов

Подпроект 1 (реализация определённого паттерна [strategy]):
~/---------/strategy/                           |-папка подпроекта (реализация 
                                                  определённого паттерна)
~/-------------------/docs/                     |-папка для документов, описаний и 
                                                  UML диаграмм подроекта
~/-------------------/build/                    |-папка для сборки подпроекта
~/-------------------------/CMakeLists.txt      |-файл для утилиты cmake
~/-------------------/src/                      |-папка для файлов реализации
                                                  подпроекта   ( *.cpp )
~/-------------------/inc/                      |-папка для заголовочных файлы 
                                                  подпроекта ( *.h )
~/-------------------/make.sh                   |-скрипт bash для сборки подпроекта
~/-------------------/clean.sh                  |-скрипт bash для отчистки исполняемых 
                                                  файлов подпроекта
                                                            ...
Подпроект n

Общие папки:
~/build_all.sh                                  |-скрипт bash для сборки всего проекта
~/clera_all.sh                                  |-скрпит bash для отчитки исполняемых 
                                                  файлов всего проекта
~/README.TXT                                    |-текстовый файл с описанием
